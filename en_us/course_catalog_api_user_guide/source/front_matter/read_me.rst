The edX *Course Catalog API User Guide* is created using RST_ files and
Sphinx_. You, the user community, can help update and revise this documentation
project on GitHub::

  https://github.com/edx/edx-documentation/tree/master/en_us/course_catalog_api_user_guide/source

To suggest a revision, fork the project, make changes in your fork, and submit
a pull request back to the original project: this is known as the `GitHub
Flow`_. All pull requests need approval from edX. For more information, contact
edX at docs@edx.org.
